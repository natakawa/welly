2. Использование Чат-бота

2.1 Использование Чат-бота регулируется положениями Соглашения, включая функции предоставления Пользователю возможности формулировать запросы для Чат-бота в рамках написания текстов, предоставления ответов на вопросы Пользователя, написания программного кода при помощи Чат-бота. Чат-бот реализован на базе LLM модели Gemini 2.0 Flash.

2.2 Администратор предоставляет Пользователю право использовать Чат-бот, Личный кабинет, сервисы Чат-бота по их прямому назначению, в том числе путём осуществления доступа к Чат-боту, Личному кабинету с помощью персональных компьютеров и мобильных устройств (в рамках веб-версии, мобильной и десктопной версий мессенджера Telegram) и использования явных функций Чат-бота, Личного кабинета на условиях безвозмездной простой (неисключительной) лицензии на территории доступа к Чат-боту, Личного кабинета и к их функциям на срок, в течение которого Чат-бот и его функции остаются доступны для Пользователя.

3. Описание продуктов и Чат-бота

3.1 Чат-бот «Welly – твой помощник по питанию» предназначен для формулирования обратной реакции искусственного интеллекта, обученного на базе общеизвестной базы знаний на запросы Пользователей в рамках предоставления информации по питанию и определению нутрициологического состава блюд, создания текстов по запросам Пользователей.

3.2 В соответствии с Соглашением и любыми другими применимыми специальными документами между Пользователем и Администратором, Чат-бот предоставляется для ответов на запросы Пользователей и выполнения их задач, сформулированных для Чат-бота.

3.3 Администратор оставляет за собой право по своему усмотрению изменять, модифицировать, добавлять или удалять функции и возможности Чат-бота в любое время.
